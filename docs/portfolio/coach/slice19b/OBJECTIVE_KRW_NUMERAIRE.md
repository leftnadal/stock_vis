# 목표의 기준통화(numéraire) = KRW — 방향 문서

> 사용자 확정(2026-07-14). 채팅에만 있던 방향 원칙의 repo 안착(유실 방지). DECISIONS `SLICE19B`가 최종 진실.

## 원칙
- **기준통화 = KRW.** "수익" = 원화 자산의 증가. 사용자 #1의 삶은 원화로 영위되므로, 투자 성과의 정본은 KRW 표시.
- **환율은 수익의 구성요소(곁다리 아님).** US 종목 실수익 = (USD 주가 수익) × (USD/KRW 변화). 달러가 강해지면 같은 USD 수익이라도 KRW 수익은 커지고, 약해지면 작아진다.
- **19a의 "통화별 사일로 갭"은 KRW 기준으로 교정한다.** 진행 갭·배치 갭·매수여력이 KRW 환산 기준으로 계산되는 것이 정본. 통화별 소계는 참고로 유지.

## 정직성 4원칙 (FX 취급)
- **측정 O**: USD 자산을 현재 환율로 KRW 평가.
- **맥락 O**: 현재 USD/KRW의 역사적 백분위(사실)를 근거에 표시.
- **실현 FX 반응 O (emergent)**: "환율 나빠지면 더 공격적"은 KRW 갭이 자동으로 벌어져 emergent — 별도 규칙 없음.
- **예측 X**: 환율 방향·타이밍 베팅 금지. 환전 결정에 예측 사용 금지.

## 취득원가(게이트 1 해소)
- 과거 KRW 실취득원가 정본은 미저장 → 우선순위 로직으로 산정: `exact`(사용자 캡처/정정) > `approx_first_buy`(매수일 환율 근사) > `approx_low_confidence`(창 밖). 상세 = `SLICE19B_GATE1_RESOLUTION.md`.
