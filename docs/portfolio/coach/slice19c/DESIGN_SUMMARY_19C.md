# Slice 19c — 배치 엔진 v2 설계 종합

> 1차 소스 = `DECISIONS.md § SLICE19C`. 이 문서는 실행 요약(지시서 서문 수준).
> 결정 확정 2026-07-14(디렉터·사용자), 실행 2026-07-16. base `ef8990c`, baseline 602.

## 왜 이 슬라이스인가
19a·19b 엔진은 임계값(현금 10%·집중도 30%)이 **하드코딩된 고정 가치관**이었다. 19c는
성향을 **사용자 주권**으로 이양하고, 자산 감소에 **비례 반응**하는 다이얼을 도입한다.
엔진의 고정 가치관은 단 둘 — **"현금 3%는 남긴다"(버퍼 바닥) + "사실은 숨기지 않는다"(사실 보고)**.

## 4대 축
1. **드로다운 비례 다이얼** (하드 10% 버퍼 게이트 대체)
   ```
   dd   = (조정 고점 − 현재 총자산 KRW) / 조정 고점      # 가격·환율 효과만 (flow 분해)
   a    = dd + A + G·𝟙(신고점 국면)
   버퍼 = max(10% − a, 3%)                              # 바닥 3% = 불가침 불변식
   여력 = 유휴현금 비중 − 버퍼                            # 통화별 현금 비례 배분, 음수=0 클램프
   ```
   - 고점 = UserGoal 스코프 관측 최대(목표 생성/변경 시 리셋). 콜드 스타트 특별 규칙 없음.
   - flow 분해: 스냅샷 diff로 가격효과(전일 수량 고정 × 가격·환율 변화)·플로우효과(입출금 잔차) 분리. dd = 가격효과 누적만. 조정 고점 = 고점 + 고점 이후 순입금.
   - 가격 신선도: 보유 가격 나이 > 영업일 2일 → dd만 직전 유효값 동결(랭킹·게이트는 정상).

2. **성향 손잡이 5종** (UserGoal 가산 필드, 전부 default = 보수)
   | 손잡이 | 필드 | 범위 | 기본 | 의미 |
   |---|---|---|---|---|
   | A | aggressiveness_offset | 0~7 (%p) | 0 | 상시 여력 오프셋 |
   | G | growth_boost | 0~7 (%p) | 0 | **신고점 국면에서만** 적용 |
   | w | diversification_weight | 0~0.20 | 0 | 코어 랭킹 분산 성분 |
   | L | concentration_limit | 15~100 (%) | 30 | 집중도 한도(자격+TRIM). 100=무제한(TRIM 소멸) |
   | E | exploration_ratio | 0~30 (%) | 0 | 여력 중 탐험 레인 배정 |
   - 엔진이 성향을 자동 조정하지 않는다. 자동 반응은 dd(측정 사실) 하나뿐.

3. **하이브리드 게이트 × 스코어** (하드 30% → L 대체)
   - 게이트=자격: 매수 시 집중도>L 제외 · 현재>L TRIM · 통화 여력 0 자격 박탈. L=100 → TRIM 소멸.
   - 코어 랭킹(소프트 서열): `점수 = (1−w)×(신뢰도 0.60 + 진입가 여유 0.25 + 통화 여력 0.15) + w×분산 한계효과`.
     분산 한계효과 = 표준 트랜치(=통화 매수여력) 가정 매수 후 집중도 개선/악화(사실 계산). w=0이면 정확히 0.
   - **신뢰도 지배 불변식**: w 상한 0.20 → 신뢰도 성분(0.48) 항상 최대. 명칭 = "배치 우선순위 점수(기대수익 아님)".

4. **탐험 레인 E**
   - 여력 × E = 젊은 후보(관측 이력 짧은, `first_observed_at < 30일` — STEP 0 (c) 자리표시) 전용.
   - 신뢰도 **무보정**(젊음 가산점 금지 — 배정만 분리). 레인 내 서열 = 진입가 여유·통화 여력. 게이트·바닥 3% 안에서만.

## 원장 2종 (사후분석 토대)
- `PortfolioSnapshot`: user·date·총자산 KRW·통화별 소계·종목별 수량/가격/환율(JSON)·가격 기준일. `unique(user,date)`. nightly+엔진 upsert 이중.
- `AdvisoryRun`: 실행시각·snapshot 참조·산출 전문(JSON)·손잡이 5종 스냅·레인(코어/탐험) 구분.

## 원칙 3계층 재정의 (기존 DISCLAIMER 정정)
- ⑴ 환율 **방향** 예측 = 영구 금지.
- ⑵ 환율 **수준** 반응 = 허용(KRW 평가 내재·emergent).
- ⑶ 종목 **수익성 예측 = 목표**(정본 신호 위·유령/프록시 금지 + 기록 후 사후분석). 구현 = `SIGNAL-FORWARD-INFRA`(19c 밖, 우선순위 상승).

## 불가침 불변식 (테스트로 증명)
버퍼 바닥 3% / 신뢰도 지배(w 상한 0.20) / 사실 보고. 어떤 손잡이 조합으로도 불가침.
