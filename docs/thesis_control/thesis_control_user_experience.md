# Thesis Control — 사용자 경험 여정 문서

> 최종 업데이트: 2026-03-30
> 상태: Phase 2 완료 + Suggestion Mode 구현

---

## 1. 개요

**가설 통제실(Thesis Control)**은 사용자가 투자 가설을 세우고, 전제를 정의하고, 관련 지표를 설정한 뒤, 시장 데이터 변화를 실시간으로 추적하며 가설의 유효성을 판단하는 기능입니다.

### 핵심 가치
- **진입 마찰 감소**: 뉴스 이슈 클릭 한 번으로 AI가 가설 2개를 제안
- **투명한 의사결정**: 가설→전제→지표 선택 과정이 Decision Trail로 시각화
- **지속적 추적**: 달/화살표 시각화로 가설 건강도를 직관적으로 파악

## 3. 화면별 상세

### 3-1. 가설 목록 (`/thesis`)

사용자가 처음 진입하는 메인 화면입니다.

#### 구성 요소

| 영역 | 내용 |
|------|------|
| 헤더 | "가설 통제실" + 알림 벨 (읽지 않은 알림 카운트 뱃지) |
| 추적 중 | 활성 가설 카드 목록 (우선순위: critical → needs_review → weakening → ...) |
| 오늘의 변화 | 최근 알림 3개 요약 (읽지 않은 것만) |
| 새 가설 | "새 가설 세우기" 파란 버튼 |

#### 가설 카드 (ThesisListCard)

```
┌─────────────────────────────────────────────┐
│  🌕   삼성전자 반도체 반등                    > │
│       📈 상승 · 강화 중 · 삼성전자 · 12일째     │
└─────────────────────────────────────────────┘
```

- **왼쪽**: 달 위상 아이콘 — 가설 건강도를 시각화
  - 🌕 밝은 보름달 (score > 0.6) → 가설이 강하게 지지됨
  - 🌔 차오르는 달 (0.2~0.6) → 지지하는 편
  - 🌗 반달 (-0.2~0.2) → 중립
  - 🌘 기우는 달 (-0.6~-0.2) → 약화하는 편
  - 🌑 그믐달 (< -0.6) → 가설이 반박됨
  - ⏳ 흐린 달 (null) → 데이터 수집 중
- **중앙**: 제목 + 방향/상태 뱃지 + 대상 + "N일째 추적 중"
- **터치**: 대시보드(`/thesis/[id]`)로 이동

#### 빈 상태
가설이 없으면 중앙에 안내 문구와 하단 "새 가설 세우기" 버튼이 표시됩니다.

### 3-2. 가설 세우기 (`/thesis/new`)

대화형 인터페이스로 가설을 구조화합니다. 전체 화면 어두운 배경(gray-950), 채팅 형식.

#### Step 1: 진입 방식 선택

```
┌─────────────────────────────────┐
│  AI: 어떤 투자 가설을 세워볼까요?  │
│                                 │
│  💬 내 생각                      │
│     자유롭게 입력해보세요          │
│                                 │
│  📰 오늘 이슈                    │
│     최근 시장 이슈에서 시작        │
│                                 │
│  ⭐ 인기 가설                    │
│     검증된 템플릿으로 시작         │
└─────────────────────────────────┘
```

3가지 진입 경로:
- **내 생각**: 자유 텍스트 입력 → AI가 가설로 구조화
- **오늘 이슈**: 뉴스 기반 이슈 카드 → AI 가설 제안 (Suggestion Mode)
- **인기 가설**: 검증된 4개 템플릿 중 선택

#### Step 2: 뉴스 이슈 선택 (오늘 이슈 경로)

```
┌─────────────────────────────────┐
│  User: 오늘 이슈에서 시작         │
│                                 │
│  AI: 최근 주요 이슈를 정리했어요.   │
│                                 │
│  ── 거시 MACRO ──               │
│  📈 유가 급등 리스크              │
│     에너지 비용 상승 영향...       │
│     원문 ↗                       │
│                                 │
│  📉 미국 금리 인하 기대           │
│     연준 비둘기 발언 영향...       │
│                                 │
│  ── 미시 MICRO ──               │
│  📰 테슬라 판매량 급감            │
│     중국 시장 점유율 하락...       │
│                                 │
│  직접 입력하기                    │
└─────────────────────────────────┘
```

- Gemini가 최근 7일 뉴스를 분석하여 투자 이슈로 변환
- 거시(Macro)/미시(Micro) 분류
- 감성 아이콘: 📈 긍정 / 📉 부정 / 📰 중립

#### Step 3: AI 가설 제안 (Suggestion Mode)

이슈를 선택하면 AI가 같은 팩트를 **상승/하락** 두 관점으로 해석한 가설 2개를 제시합니다.

```
┌────────────────────┐ ┌────────────────────┐
│ ↑ 상승              │ │ ↓ 하락              │
│                    │ │                    │
│ 유가 상승으로        │ │ 유가 상승이          │
│ 에너지 기업 수혜     │ │ 기업 마진 압박       │
│                    │ │                    │
│ EU 규제 강화에도     │ │ 원자재 비용 증가로    │
│ Meta의 일평균 사용자  │ │ 제조업 이익률이...    │
│ 32억 명은...        │ │                    │
│                    │ │                    │
│ ── 근거 ──          │ │ ── 근거 ──          │
│ • 원유 수급 타이트   │ │ • 운송비 상승 압박   │
│   OPEC 감산 영향... │ │   물류비 증가로...   │
│ • 달러 약세 기조     │ │ • 소비 위축 가능성   │
│   금리 인하 기대...  │ │   실질 소득 감소...  │
│                    │ │                    │
│ [이 가설로 시작 →]   │ │ [이 가설로 시작 →]   │
└────────────────────┘ └────────────────────┘

직접 작성하기
```

**핵심 설계 원칙**:
- 제목: 20~40자, 구체적 주장
- 요약: 2~3문장, "팩트→해석→결론" 논리 흐름
- 전제: 각 15~30자 + 구체적 근거/수치 포함
- 추상적 문구 금지 ("규제 영향 제한적" ❌ → "EU 벌금은 매출의 1% 미만" ✅)

#### Step 4: 전제 선택 (Decision Trail)

카드를 선택하면 해당 가설의 전제가 체크박스로 표시됩니다. 기본 전체 선택이며, 사용자가 불필요한 전제를 해제합니다.

```
┌─────────────────────────────────────┐
│  ┌─ ✓ 선택됨 ────┐ ┌─ (흐림) ────┐ │  ← Decision Trail:
│  │ 유가 수혜      │ │  (opacity    │ │     선택/미선택 시각 구분
│  │ • 원유 수급... │ │   35%)      │ │
│  └───────────────┘ └──────────────┘ │
│                                     │
│  AI: 전제를 선택해주세요              │
│                                     │
│  ☑ 원유 수급 타이트                  │  ← 전체 pre-select
│  ☑ 달러 약세 기조                    │
│  ☐ 에너지 수요 회복 (해제 가능)       │
│                                     │
│       ─── 선택 완료 (2개) → ───      │
└─────────────────────────────────────┘
```

**Decision Trail**: 이전 단계의 선택 결과가 비활성 상태로 스크롤 이력에 남습니다.
- 선택된 카드: 풀 컬러 + "✓ 선택됨" 뱃지, 축소 표시
- 미선택 카드: opacity 35%, CTA 버튼 제거

#### Step 5: 지표 추천 + 프리셋 선택

전제 확인 후, 선택된 전제를 기반으로 AI가 지표를 매칭합니다.

```
┌─────────────────────────────────────┐
│  ┌ 선택한 전제 ──────────────┐       │  ← Decision Trail:
│  │ ✓ 원유 수급 타이트         │       │     전제 선택 결과
│  │ ✓ 달러 약세 기조           │       │
│  │ ✗ 에너지 수요 회복 (취소선) │       │
│  └───────────────────────────┘       │
│                                     │
│  AI: 전제 2개, 추천 지표 3개         │
│                                     │
│  ☑ 원유 (WTI)           일간        │
│  ┌ WTI 선물 가격이 배럴당        ┐   │  ← 추천 이유 박스
│  │ $80을 돌파하면 에너지 비용... │   │
│  └──────────────────────────────┘   │
│  전제: 원유 수급 타이트              │
│                                     │
│  ☑ 달러 인덱스 (DXY)    일간        │
│  ┌ DXY 하락은 원자재 가격          ┐ │
│  │ 상승 압력으로 작용...           │ │
│  └──────────────────────────────┘   │
│  전제: 달러 약세 기조                │
│                                     │
│  [+ 지표 추가]                       │
│                                     │
│  ─── 모니터링 기간 선택 ───          │
│  ⚡ 단기(1개월)                      │
│  📈 중기(1~3개월)                    │
│  🔍 장기(6개월+)                     │
└─────────────────────────────────────┘
```

**지표 추천 카드 (IndicatorCard)**:
- 체크박스로 개별 on/off
- 선행/동행/후행 뱃지 (파랑/회색/주황)
- 추천 이유 (회색 박스)
- 연결된 전제명

**지표 추가 시트 (+ 지표 추가 클릭 시)**:

```
┌─────────────────────────────────┐
│  지표 추가                    ✕  │
│                                 │
│  ✨ 전제 관련 추천                │
│  ┌──────────────────────────┐   │
│  │ 원유 (WTI)        일간 + │   │

│  ├──────────────────────────┤   │
│  │ VIX (공포지수)    일간 + │   │
│  │ 「달러 약세 기조」          │   │
│  │ 시장 심리/불확실성 감지     │   │
│  └──────────────────────────┘   │
│                                 │
│  ── 수급 ──                      │
│  외국인 순매수  일간  +           │
│  기관 순매수    일간  +           │
│                                 │
│  ── 주요 지수 ──                 │
│  S&P 500       일간  +           │
│  ...                             │
```

- 상단에 **전제 기반 추천** 섹션 (매칭된 전제 + 이유 표시)
- 모든 지표에 **업데이트 주기 뱃지** (일간=초록, 주간=파랑, 월간=주황, 분기=회색)
- 72개 지표: 시장(27) + 거시(11) + 기술(9) + 펀더멘털(11) + 재무체질(14) + 심리(1)

**프리셋 선택**:
- ⚡ 단기 (1개월): 빠른 변화 감지, 민감한 알림
- 📈 중기 (1~3개월): 균형 잡힌 추적
- 🔍 장기 (6개월+): 중요 변화만 알림

## 4. 상태 시각 체계

### 가설 상태 (달 위상)

| 점수 | 아이콘 | 상태 | 의미 |
|------|--------|------|------|
| > 0.6 | 🌕 | full_moon | 가설이 강하게 지지됨 |
| 0.2~0.6 | 🌔 | waxing | 지지하는 편 |
| -0.2~0.2 | 🌗 | half_moon | 중립 |
| -0.6~-0.2 | 🌘 | waning | 약화하는 편 |
| < -0.6 | 🌑 | new_moon | 가설이 반박됨 |
| null | ⏳ | warming_up | 데이터 수집 중 |

### 지표 방향 (화살표)

0°~180° 범위의 화살표 각도로 지표가 가설을 지지/반박하는 정도를 표현합니다.
- 0° (→): 강하게 지지
- 90° (↑): 중립
- 180° (←): 강하게 반박
