## 1. Executive Summary

### 1.1 Stock-Vis의 위치 정의

**Stock-Vis**는 AI 기반 투자 분석 플랫폼으로, 초보 개인투자자부터 중급 퀀트 투자자를 타깃으로 하는 **교육적 투자 플랫폼**입니다.

현재 상태:
- **개발 단계**: Phase 3 완료 (2026년 1월)
- **핵심 강점**: RAG 기반 AI 분석, 거시경제 대시보드, 관심종목 관리
- **데이터 충실도**: 중상 (필수 요소 70~80% 커버)

### 1.2 Phase 3 완료 후 핵심 변화

| 항목 | Before (Phase 2) | After (Phase 3) |
|-----|-----------------|-----------------|
| **개별주식 데이터** | Alpha Vantage만 | Alpha Vantage + FMP + yfinance |
| **거시경제** | FRED, CNN만 | FRED + yfinance + 실시간 공포/탐욕 지수 |
| **시장 선택지표** | 개별 종목만 | Market Movers (Gainers/Losers/Actives) + 5개 고급 지표 |
| **AI 분석 능력** | 기본 분석만 | 의미론적 캐시 + 복잡도 분류 + 적응형 모델 선택 |
| **사용자 경험** | 기본 화면 | 실시간 포트폴리오 + 관심종목 목표가 추적 + Watchlist |
| **비용 효율성** | 월 $14 (FMP) | 월 $34 (FMP $14 + EODHD $19.99) |

### 1.3 경쟁 우위 요약

**Stock-Vis만의 차별화 포인트:**

1. **교육적 AI 분석**: 단순 추천이 아닌 "왜"를 설명하는 RAG 분석
   - 초보자가 투자를 배우면서 실전 경험
   - 전문가의 도움 받는 느낌

2. **한국 시장 최적화** (향후)
   - KOSPI/KOSDAQ 데이터 (현재는 미국 시장만)
   - 한국어 뉴스 & 센티먼트

3. **Market Movers 고급 지표 시스템**
   - Phase 1: RVOL, Trend Strength (기본)
   - Phase 2: Sector Alpha, ETF Sync Rate, Volatility Percentile
   - Phase 3: 자동 거래 신호 (향후)

4. **통합된 거시경제 분석**
   - 개별 종목 분석 + 시장 전체 컨텍스트
   - Market Pulse: Fear & Greed Index + 금리곡선 + 글로벌 시장

### 1.4 목표 사용자층과 시장 포지셔닝

| 페르소나 | TAM (Total Addressable Market) | Stock-Vis 포지셔닝 |
|---------|--------------------------------|-------------------|
| **초보 개인투자자** | 한국 2,000만 MZ세대 | 차트 읽기부터 기술적 분석까지 배우기 |
| **중급 개미투자자** | 한국 200만 활동 투자자 | 데이터 기반 의사결정 지원 |
| **준전문가 (반딩이)** | 한국 20만 유명 투자자 | AI 기반 시나리오 분석 |
| **대학 금융학과** | 한국 200개 대학, 10만 학생 | 투자 교육 도구 |

**초기 타깃**: 초보 + 중급 개인투자자 (한국 500만 명, 3%~5% 침투율 목표 = 15만~25만 사용자)

### 3.4 AI/RAG 분석 능력

#### 3.4.1 현재 RAG Phase 3의 강점

| 기능 | 구현 상태 | 평가 |
|-----|---------|------|

| **비용 추적** | ✅ | 사용자/날짜별 로깅 |

1. **비용 최적화**: 불필요한 고급 모델 사용 방지 → 40% 비용 절감
2. **응답 속도**: Simple 질문은 Haiku로 2초 이내 응답
3. **맥락 이해**: RAG + Semantic Cache로 연관 분석 정확도 85%+

#### 3.4.2 새로운 데이터 통합 후 개선 가능성

| **옵션 데이터** | 시장 센티먼트 분석 | "기관은 뭔가 알고 있나?" 기관 의도 해석 |

   - Sector Alpha + ETF Sync + Volatility %ile은 Stock-Vis만 제공
   - 중급 투자자의 의사결정 가속화

3. **통합 대시보드** (Market Pulse)
   - 개별 종목 + 거시경제 + 포트폴리오 한 화면에
   - 초보자가 "전체 시장을 본다"는 느낌
