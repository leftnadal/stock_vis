## 2️⃣ 시그널 체계 검토

### 2-1. Tier 1 (즉시 구현) — 강점

#### 가격 움직임

| 시그널 | 평가 | 코멘트 |
|--------|------|--------|
| **P1 연속 상승/하락** | ⭐⭐⭐⭐⭐ | 초보자 입장에서 가장 직관적. "5일 연속 상승" = 모멘텀 입문 |
| **P2 수익률 상위** | ⭐⭐⭐⭐⭐ | "오늘의 로켓 🚀" 명명이 탁월. FOMO 방지 교육 필수 |
| **P3 갭 감지** | ⭐⭐⭐⭐ | 좋음. "점프" = 뉴스 반응 포착. 단기 변동성 기대 설정 필요 |
| **P4 장대양봉** | ⭐⭐⭐⭐ | 강력한 신호. 초보자도 캔들 모양으로 이해 가능 |
| **P5 52주 신고가 근접** | ⭐⭐⭐⭐⭐ | 심리적 지지점. 돌파 시 추진력 있음. 전문적 |
| **P6 라운드 피겨** | ⭐⭐⭐ | 흥미로우나 **투자 판단 실질성 낮음**. 심리적/기술적 마디가 실제로 중요한지 실증 필요. 우선순위 낮춤 권장 |
| **P7 저가 대비 반등률** | ⭐⭐⭐⭐⭐ | 매우 좋음. 리스크/리워드 비율을 암시. "장중 폭락 후 살아 돌아온" = 강한 수요 신호 |

**결론**: P6를 제외하고는 매우 우수. **P6은 Phase 1에서 제외하고 Phase 3에서 검증 후 재검토 권장**.

---

#### 거래량

| 시그널 | 평가 | 코멘트 |
|--------|------|--------|
| **V1 거래량 이상 감지** | ⭐⭐⭐⭐⭐ | 핵심 신호. "평소의 3배" = 객관적, 해석 명확 |
| **V2 신규 거래 활성** | ⭐⭐⭐⭐ | 좋음. 저유동성 종목 모니터링에 유용. 초보자에게는 V1만으로도 충분할 수 있음 |
| **V3 비정상적 변동성** | ⭐⭐⭐ | **다소 모호**. "변동성 높음" = "기회" vs "위험"이 불명확. 맥락이 필요함 |

**권장 사항**:
- **V1, V2**: 메인. V1은 항상 상위 시그널.
- **V3**: Phase 1에서 제외. Phase 2에서 "보합장 중 급등락 = 이상 신호" 맥락으로 재정의 후 사용.

#### 가격-거래량 복합 (돈의 흐름)

| 시그널 | 평가 | 코멘트 |
|--------|------|--------|
| **PV1 가격-거래량 효율성** | ⭐⭐⭐⭐⭐ | "가볍게 날아오르는" 은유가 완벽. 기술적으로도 유효 (구매력 높음) |
| **PV2 매집 의심** | ⭐⭐⭐⭐⭐ | 초보자 입장에서 "누군가 아래에서 받아먹고 있다" = 마음이 설레는 시그널. 실증 필요하지만 흥미로움 |
| **PV3 장 마감 매집** | ⭐⭐⭐ | **흥미롭지만 신뢰성 미확인**. 3일 연속 판단이 과도한가? 패턴이 실제로 선행 수익률 있는지 검증 필수. Phase 2 이후 추가 권장 |

**권장 사항**:
- **PV1, PV2**: Phase 1 포함.
- **PV3**: Phase 2에서 검증 후 추가.

### 2-2. Tier 2 (수일~수주 히스토리) — 평가

#### 이동평균 & 추세

| 시그널 | 평가 | 코멘트 |
|--------|------|--------|
| **MA1 골든/데드크로스** | ⭐⭐⭐⭐⭐ | 고전이자 강력. 초보자도 이해 쉬움. 모멘텀 확인의 표준 |
| **MA2 이동평균 이격도** | ⭐⭐⭐⭐ | "과열 주의" 메시지 좋음. 과도한 상승 후 조정 신호 |
| **MA3 이동평균 수렴** | ⭐⭐⭐⭐ | "곧 방향이 정해진다" = 유용한 대기 신호. 진입/관찰의 경계 |
| **MA4 추세 전환 감지** | ⭐⭐⭐⭐⭐ | "하락→상승 전환" = 역발상 투자자에게 매력적 |

**평가**: 모두 실질적이고 유용함. 단, **초보자에게는 MA1, MA4만 우선 노출하고, MA2/MA3는 "기술적 지표" 심화 섹션에 배치 권장**.

#### 패턴

| 시그널 | 평가 | 코멘트 |
|--------|------|--------|
| **PT1 지지/저항선 돌파** | ⭐⭐⭐⭐⭐ | 60일 최고가 갱신은 강한 신호. 기술적으로 정당화됨 |
| **PT2 가격 압축** | ⭐⭐⭐⭐ | "조용하다가 곧 터질 수 있다" 매력적이지만, **임계값이 "20일 표준편차 역대 하위 10%"로 너무 엄격할 수 있음**. 상위 20%로 완화 권장. Phase 2 검증 필수 |
| **PT3 N일 만에 첫 양봉** | ⭐⭐⭐⭐⭐ | "5일 연속 하락 후 첫 반등" = 극도의 공포 이후 회복. 매우 좋은 신호 |
| **PT4 바닥 다지기** | ⭐⭐⭐⭐ | "복수혈전" 은유가 훌륭함. 다만 "60일 신저가 후 5일 저가 미갱신"이 바닥 증거로 충분한가? 부피(거래량)가 증가했는가 확인 권장 |
| **PT5 전고점 매물대 소화** | ⭐⭐⭐⭐ | 기술적으로 유효. "매물대 극복" = 심리적 돌파감 |
| **PT6 거래량 동반 상승** | ⭐⭐⭐⭐⭐ | 매우 좋음. "돈이 몰리면서 오르는" = 지속성 신호. 강추 |

**권장 사항**:
- **PT1, PT3, PT5, PT6**: Phase 1 포함.
- **PT2, PT4**: Phase 2에서 임계값 검증 후 추가.

#### 섹터/그룹 상대 강도 → Chain Sight 직접 연결

| 시그널 | 평가 | 코멘트 |
|--------|------|--------|
| **S1 섹터 상대 강도** | ⭐⭐⭐⭐⭐ | **핵심 신호**. Chain Sight 입문의 최고 포인트. "혼자 버티는 종목" = 뭔가 다른 이유가 있다 → 탐색 욕구 자극 |
| **S2 섹터 소외주** | ⭐⭐⭐⭐⭐ | "친구들은 다 갔는데" = 심리적 FOMO 역이용. 포착하기 쉬운 불균형. Chain Sight 진입 최고의 신호 |
| **S3 대장주 따라잡기** | ⭐⭐⭐⭐ | 좋지만 S2와 개념 유사. 우선순위 조정 (S2 > S3) 권장 |
| **S4 폭락장 생존자** | ⭐⭐⭐⭐⭐ | SPY -2% 이상 하락장에서 보합/상승 = 매우 우수한 방어력 신호. 위험 선호도 낮은 투자자에게 어필 |

**평가**: 매우 좋음. **S1, S2, S4는 필수. S3은 우선순위 낮춤 권장**.

### 2-4. 종합 시그널 분류 재검토

#### 현재 분류

```
momentum    — P1, P2, P3, P4
volume      — V1, V2, V3, PV1, PV2, PV3
breakout    — P5, P6, PT1, PT5
reversal    — P7, PT3, PT4, A4
relation    — S1, S2, S3, S4
technical   — MA1~4, T1~4
composite   — A1, A2, A3
```

#### 투자 관점의 개선안

```
✅ RECOMMENDED CATEGORIZATION:

momentum (즉시 판단)
  → P1 연속상승, P2 수익률상위, P3 갭, P4 장대양봉
  → 초보자 입장에서 "오늘 뭐 볼까?" 답변

volume (수요 신호)
  → V1 거래량3배, V2 신규활성, PV1 효율성, PV2 매집의심
  → "관심이 높아졌다" 해석

breakout (이격/저항 극복)
  → P5 52주신고가, PT1 저항돌파, PT5 전고점소화, PT6 동반상승
  → "다음 단계로" 진출 신호

reversal (극도의 공포 후 회복)
  → P7 저가반등, PT3 첫양봉, PT4 바닥다지기, MA4 추세전환
  → 역발상 투자자 입장

relation (관계 탐색)
  → S1 상대강도, S2 섹터소외, S4 폭락생존
  → Chain Sight 진입점

technical (심화 학습)
  → MA1 골든크로스, T1 RSI, T2 볼린저, T3 MACD
  → 기술적 지표 입문

advanced (고급 신호)
  → A1 OBV, A2 멀티타임프레임, N1 시간차공격
  → Phase 2 이후 노출

⚠️ EXCLUDE / DEFER:
  → P6 라운드피겨 (검증 필요)
  → V3 변동성 (맥락 불명확)
  → PV3 장마감매집 (임계값 미검증)
  → PT2 압축 (임계값 과도)
  → S3 대장따라잡기 (S2와 중복)
  → T4 ADX (초보자 복잡)
  → A3 변동성조정 (고급)
```

## 3️⃣ 화면 구성 검토

### 3-1. 메인 대시보드 (Cold Start 진입점)

#### 강점

| 요소 | 평가 | 이유 |
|------|------|------|
| **시장 요약 바** | ⭐⭐⭐⭐⭐ | SPY, QQQ, VIX 한눈에. 초보자도 "오늘 시장이 어떤가" 즉시 파악 |
| **시그널 카드 그리드** | ⭐⭐⭐⭐⭐ | 수평 스크롤로 다양한 시그널을 "시장의 여러 신호" 형태로 표현. 직관적 |
| **카드당 Top 3-5 종목** | ⭐⭐⭐⭐⭐ | 초보자를 압도하지 않음. "오늘은 이 3개만 보자" 선택지 제공 |
| **카드별 교육 팁** | ⭐⭐⭐⭐⭐ | "거래량은 시장 관심의 크기" 설명 좋음. 매 방문마다 학습 |

#### 개선 사항

| 구성 요소 | 현재 | 문제점 | 제안 |
|---------|------|--------|------|
| **시그널 카드 우선순위 표시** | 건수 순 | 초보자에게 "건수 = 중요도"가 아닐 수 있음 | Phase 2: 클릭 빈도 + 정확도(signal_accuracy) 기반 가중치로 개인화 정렬. Phase 1은 고정 우선순위 (momentum > volume > reversal > breakout) 권장 |
| **뉴스 하이라이트** | 동적 계산 필요 | | 좋은 구성. 단, "시그널 → 뉴스" 순으로 표현하여 인과관계 명확히 |
| **Chain Sight 미니맵** | 미정 (Phase 2) | | 찬성. 하지만 초보자에게는 "관계 탐색" 레이어가 복잡할 수 있음. 선택형 / 펼침형으로 제공 권장 |
| **시그널 카드 색상** | 미정 | | **각 카드별 일관된 색상 코드 권장**. momentum = 🔥빨강, volume = 💥노랑, reversal = 💪초록, breakout = 📈파랑 등 시각적 일관성 |
| **"유사 패턴도 함께" CTA** | 없음 | 초보자가 한 종목에만 집중하기 쉬움 | "이 종목 왜 좋을까?" → Chain Sight 진입 CTA 버튼 추가. "NVDA 거래량 폭발 — AMD는?" 유도 |

#### 초보자 관점 테스트 시나리오

```
시나리오 1: 수익률 상위 카드 (P2)
현 상태: "오늘의 로켓 🚀 12종목"
문제: 초보자가 이 12개 중 뭘 골라야 하나 막연해함
개선:
  ① 거래량도 동반 상승했는가? (확인)
  ② 같은 섹터의 다른 종목도 올랐는가? (관계 확인)
  ③ 뉴스는 있는가? (이유 확인)
  → 이 3가지 위젯으로 선택 신뢰도 향상

시나리오 2: 섹터 소외주 카드 (S2)
현 상태: "친구들은 다 갔는데 아직 출발 안 한 종목 7종목"
문제: 정말 "따라갈 종목"일까? 아니면 "뒤처진 이유가 있는 종목"일까?
개선:
  ① "왜 뒤처졌는가?" Chain Sight로 바로 진입
  ② 해당 회사 프로필/최근 이슈 표시
  ③ 초보자는 "따라갈 확률 높음 ✅" vs "주의 필요 ⚠️" 분류 확인
  → signal_accuracy 데이터 기반으로 이전 유사 신호의 성과도 표시
```

### 3-4. Cold Start → Portfolio 플로우

#### 강점

```
메인 대시보드 → 시그널 카드 → 개별 종목 → Chain Sight → 모니터링 → 포트폴리오
```

이 흐름이 **초보자가 시장을 이해하고 판단하는 과정과 정확히 일치**한다는 점이 탁월함.

**단계별 학습 효과**:
1. **메인 대시보드**: "오늘 시장이 어떤가" (시장 파악)
2. **시그널 카드**: "특정 패턴에 해당하는 종목 발견" (패턴 인식)
3. **종목 상세**: "이 종목이 왜 선택되었는가" (개별 분석)
4. **Chain Sight**: "이 종목이 왜 중요한가, 연관된 기회는" (관계 이해)
5. **모니터링**: "지속적 추적으로 신뢰도 확보" (의사결정 강화)
6. **포트폴리오**: "실제 투자 결정" (실행)

#### 개선 사항

| 단계 | 현재 | 문제점 | 제안 |
|------|------|--------|------|
| 1→2 전환 | 카드 클릭 | 명확함 | 유지 |
| 2→3 전환 | 종목 클릭 또는 [Chain Sight] | 경로 2개 | Phase 1: 종목 클릭 우선. Phase 2: [관련 종목 보기] CTA 추가로 Chain Sight 직진 유도도 가능하게 |
| 3→4 전환 | [Chain Sight 🔗] | 명확함 | 유지. 단, 초보자에게는 "왜 Chain Sight인가" 1줄 설명 추가 ("관계를 보면 더 많은 기회를 발견할 수 있어요") |
| 4→5 전환 | [모니터링+] | 명확함 | Chain Sight에서 발견한 종목도 바로 [모니터링+] 가능하게. 경로 단축 |
| 5→6 전환 | 포트폴리오 진입 | 모니터링만으로는 투자 안 함. 추가 신호 필요 | 모니터링 종목의 시그널 변화 알림 + "오늘 새로운 신호 발생! ✨" 스타일의 푸시 알림 → 포트폴리오 추가 유도 |

#### 초보자 심리 상태 매핑

```
메인 대시보드
├─ "오늘 뭐 할까?" → 시장 파악 욕구
├─ "이 신호들 중 뭐가 진짜지?" → 검증 욕구
│
시그널 카드
├─ "아 이런 패턴 있구나" → 학습
├─ "그럼 이 종목은?" → 호기심
│
종목 상세
├─ "이 종목 최근 뭐 했어?" → 히스토리 확인
├─ "근데 왜 특별한데?" → 맥락 찾기
│
Chain Sight
├─ "아 이렇게 연결되어 있구나" → 관계 이해
├─ "그럼 AMD도 볼 가치 있겠네?" → 추가 기회 발견
│
모니터링 등록
├─ "매일 이 종목 신호 변할 때마다 알려줘" → 신뢰도 축적
│
포트폴리오 추가
├─ "확신이 생겼으니까 투자할게" → 의사결정 완성

✅ 이 흐름은 **신뢰도 축적의 자연스러운 진행**이다. 잘 설계됨.

⚠️ 개선점:
  - 모니터링 → 포트폴리오 전환이 자동이 아님
  - "이 종목 정말 좋을까?" 최종 확신 신호 필요
  - Phase 3: 모니터링 종목의 "신뢰도 스코어" 시각화 추천
```

⚠️ 주의 메시지:

### 최종 의견

> **이 설계는 매우 우수하다.**
>
> 초보 투자자가 "시장을 이해하고, 신뢰도 있게 판단하고, 관계를 탐색하는" 완전한 투자 학습 여정을 지원한다. 특히:
>
> 1. **시그널 스토리텔링**: "로켓 🚀", "폭발 💥", "친구들은 다 갔는데 🐢" 같은 명명이 기억력과 공감도를 높인다.
> 2. **흐름의 자연스러움**: 시장 파악 → 신호 발견 → 종목 선택 → 관계 탐색 → 의사결정이 전문가의 실제 프로세스와 일치한다.
> 3. **뉴스와 신호의 역할 재정의**: "선 시그널, 후 뉴스"라는 철학이 초보자에게 전문적 안목을 심어준다.
> 4. **단계적 진입**: Cold Start부터 포트폴리오까지 경로가 명확하고, 언제든 후퇴 가능한 구조다.
>
> **개선점은 미미하다**: 신뢰도 표시, 경고 메시지 강화, 기술 용어 설명 정도면 충분하다.
>
> **리스크**: 초보자가 신호를 과신하거나 "모든 신호 따라하기"에 빠질 가능성. **해결책**: Phase 1부터 "이 신호의 한계는" 문구, 경고 메시지, 신뢰도 표시로 신중함을 강조한다.
>
> **이 대시보드는 초보 투자자를 전문 투자자로 성장시키는 "학습 플랫폼"이 될 수 있다.**
