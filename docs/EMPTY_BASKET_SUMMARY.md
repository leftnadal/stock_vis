# 빈 바스켓 대응 전략 - 최종 요약

**작성**: 투자 도메인 전문가 (Investment Advisor)
**날짜**: 2025-12-15
**상태**: 검토 및 구현 준비 완료

---

## 개요

Stock-Vis의 AI 분석 시스템에서 **데이터를 추가하지 않은 빈 바스켓 상태**의 사용자에게 어떻게 응답할 것인가에 대한 종합 가이드입니다.

### 핵심 원칙 3가지

```
1. 정직성 (Honesty)
   "지금은 이 정도만 가능합니다"를 명확히 전달

2. 가치 제공 (Value)
   "하지만 이렇게 도움이 됩니다"를 구체화

3. 행동 유도 (Action)
   "이렇게 하면 더 좋아집니다"를 제시

결과: 초보 투자자가 좌절하지 않고 다음 단계로 나아감
```

---

## 빈 바스켓 3가지 상황

### 상황 A: 완전 빈 바스켓

**사용자 상태**: 데이터 없음 + 질문 없음 (또는 모호함)

**응답 전략**:
- 현재 상황 인정: "분석할 데이터가 없네요"
- 가치 제시: Stock-Vis의 3가지 장점 구체화
- 명확한 다음 단계: "종목 검색 → 바구니에 추가 → 분석 시작"
- 인센티브: 데이터 추가 후 어떤 분석이 가능한지

**예시 응답**:
```
안녕하세요! 현재 분석 바구니가 비어있네요.
하지만 지금이 좋은 시작점입니다.

[Stock-Vis의 3가지 가치]
1. 여러 종목을 함께 분석해 투자 기회 발견
2. 재무 지표와 시장 흐름을 이해하기 쉽게
3. AI가 숨겨진 연결고리를 찾아줍니다 (공급망, 경쟁 관계)

[지금 할 수 있는 것]
1. 메인 화면의 "종목 검색"에서 관심 회사 선택
2. 1-3개 종목을 바구니에 추가
3. "분석 시작" 클릭 → AI가 종합 분석 제시

예) "기술주 투자를 고민 중" → Apple, NVIDIA, Microsoft 추가
   → AI가 "이들의 경쟁 관계", "공통 위험 요소" 분석

지금 관심 있는 종목 1개부터 시작해보세요!

※ 본 분석은 정보 제공 목적이며, 투자 조언이 아닙니다.
```

---

### 상황 B: 특정 질문 + 부분 데이터

**사용자 상태**: 데이터 없음 + 특정 질문 (예: "Apple은 좋은 종목인가요?")

**응답 전략**:
- 질문 이해 확인: "좋은 질문입니다"
- 데이터 없는 한계 명시: "구체적 분석은 데이터가 필요합니다"
- 가능한 설명 제공: "좋은 종목의 의미는..."
- 다음 단계: "바구니에 Apple 추가하면 이렇게 분석할 수 있습니다"

**예시 응답**:
```
좋은 질문입니다. "좋은 종목"의 기준부터 명확히 해보겠습니다.

[기준은?]
- 가치투자자: 저평가 종목 (PER, PBR 낮음)
- 성장투자자: 빠른 성장 (매출/이익 증가율)
- 배당투자자: 안정적 배당 수익
- 모멘텀 투자자: 최근 강세

모두 다른 기준이죠.

[Apple은?]
- 기술 혁신: iPhone, AI (성장 포인트)
- 브랜드 가치: 프리미엄 이미지 (방어력)
- 마진율: 높은 이익률 (효율성)

[당신의 기준을 알아야 합니다]
- "안정적인 배당이 필요합니까?"
- "5년 이상 장기 보유할 계획입니까?"
- "위험을 감수하고 성장성을 노릴 것입니까?"

[더 정확한 분석을 원한다면?]
1. 메인 화면의 "종목 검색"에서 Apple 입력
2. "바구니에 추가" 클릭
3. "분석 시작" → AI가 Apple의 재무, 경쟁사 비교 분석

※ 본 설명은 정보 제공 목적이며, 투자 조언이 아닙니다.
```

---

### 상황 C: 일반 투자 개념 질문

**사용자 상태**: 데이터 불필요 + 교육적 질문 (예: "PER이 뭐예요?")

**응답 전략**:
- 3단계 설명: 초급 → 중급 → 고급
- 구체적 실제 사례: "예를 들어 Apple과 Microsoft를..."
- 실전 연결: "Stock-Vis에서는 어떻게 활용할까?"
- 학습 → 실습: 배운 개념을 바구니에서 직접 적용

**예시 응답**:
```
좋은 질문입니다. 투자의 가장 기본이자 가장 중요한 지표니까요.

[초급: 쉽게 이해하기]
PER은 "이 회사를 사는데 몇 년치 이익을 내야 하나?"를 보여줍니다.

비유로: 카페를 10억 원에 사는데 매년 이익이 5천만 원
→ PER = 20배 = "20년 이익으로 이 카페를 살 수 있다"

[중급: 어떻게 해석할까?]
PER 낮음 = 저평가 (싼 가격에 구매 가능)
PER 높음 = 고평가 또는 고성장 기대

업계 평균 기준:
- 제조업: PER 8-15배
- 기술주: PER 20-40배
- IT 성장주: PER 50배 이상

[고급: 주의할 점]
- PER만으로는 판단 불충분 (ROE, 부채비율도 봐야 함)
- 과거 이익 기반 (미래가 다를 수 있음)
- 산업마다 기준이 다름

[실전에서는?]
1. Stock-Vis 바구니에 회사 추가 (예: Apple, Microsoft)
2. AI 분석에서 각 회사의 실제 PER 비교
3. "이 회사는 비싼가 싼가?" 직접 판단

관심 회사 1-2개를 추가해보세요. 직접 PER을 비교하는 경험이
가장 좋습니다!

※ 본 설명은 정보 제공 목적이며, 투자 조언이 아닙니다.
```

## 구현 3단계

### Step 1: 상황 판별

**ContextDetector** (Python 클래스)
- 사용자 질문 분석
- 바구니 상태 확인
- 3가지 상황 중 하나 분류

```python
reason = ContextDetector.detect_empty_reason(
    question="Apple이 좋은가요?",
    has_basket_items=False
)
# → EmptyBasketReason.PARTIAL_QUESTION
```

### Step 2: 프롬프트 선택

**PromptManager** (Python 클래스)
- 상황에 맞는 시스템 프롬프트 선택
- 가이드 데이터 생성 (UI용)
- 면책조항 추가

```python
prompt = PromptManager.get_system_prompt(
    question="PER이 뭐예요?",
    has_basket_items=False
)
# → SYSTEM_PROMPT_GENERAL_EDUCATION

guidance = PromptManager.get_guidance_data(reason)
# → {'type': 'general_education', 'title': '...', ...}
```

### Step 3: UI 표시

**EmptyBasketGuide** (React 컴포넌트)
- 가이드 메시지 표시
- 단계별 튜토리얼
- "종목 추가하기" CTA 버튼
- 면책조항

✅ 한계 명시
   "재무제표 정보가 없어 완전한 분석은 어렵습니다"

✅ 대안 제시

- 데이터 추가율: 50% (500명) ↑ 250%

## 자주 묻는 질문

### Q1: 데이터가 있는데 분석 중 데이터가 부족하면?

A: 이는 상황 B (PARTIAL_DATA)입니다.
- 있는 데이터로 분석
- 부족함을 명시: "재무제표가 없어서..."
- 필요한 것을 제시: "이 데이터가 있으면..."

### Q2: 법적 리스크는 없을까?

A: **핵심은 "정보"와 "조언"의 구분**
- "PER이 낮으면 일반적으로 저평가로 봅니다" ✅ (정보)
- "이 종목을 사세요" ❌ (조언)

**면책조항 + 명확한 한계 명시 = 법적 위험 최소화**

### Q3: 초보자가 답답해하지 않을까?

A: 오히려 반대입니다.
- 명확한 가이드 환영 ("3단계를 따라하면 됩니다")
- 가치를 느낄 때 행동 ("이렇게 하면 AI가 분석해줍니다")
- 정직한 한계를 더 신뢰 (과장된 분석보다)

### Q4: 여러 데이터 추가 후에는?

A: 그때는 바구니가 있으므로 **정상 분석 모드**로 전환됩니다.
- 상황 판별: NORMAL
- 프롬프트: SYSTEM_PROMPT_BASE
- 파이프라인: 일반 분석 로직
