**제외 권고.** 근거 2가지:
1. **일 캡 접촉**: 424B2는 하루(07-02)에만 100건 = **일 단위 창으로도 캡 도달** → 반일 창/페이지네이션 필요, 수집 복잡도 상승.

→ C2b 증자 성분 = **424B5 단독** 권고. 424B2는 §13 백로그(SEC β 후 재평가).
